# Architecture Overview

ResolveNet is a Mega Agent platform built on AgentScope and Higress.

## Components

- **Platform Services (Go)**: API server, registries, configuration management
- **Agent Runtime (Python)**: Agent execution engine, Intelligent Selector
- **WebUI (React)**: Management console with FTA workflow visual editor
- **CLI/TUI (Go)**: Command-line and terminal UI management
