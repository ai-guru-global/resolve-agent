## 什么是 ResolveNet？

ResolveNet 是一个 **CNCF 级别的开源 Mega Agent 平台**，将 **Agent 技能（Skills）**、**故障树分析（FTA）工作流** 和 **检索增强生成（RAG）** 统一在单一的智能路由层下。

## 技术栈

| 组件 | 语言/框架 | 说明 |
|------|-----------|------|
| 平台服务 | Go 1.22+ | REST/gRPC API、注册表、事件总线 |
| Agent 运行时 | Python 3.11+ | Agent 执行、智能选择器、FTA/Skills/RAG |
| CLI/TUI | Go + Bubbletea | 命令行界面与终端仪表板 |
| WebUI | React + TypeScript | 管理控制台，FTA 可视化编辑器 |
| 网关 | Higress | AI 网关，认证、限流、模型路由 |
| 数据库 | PostgreSQL | 持久化存储 |
| 缓存 | Redis | 会话管理、缓存 |
| 消息队列 | NATS | 事件驱动通信 |
| 向量数据库 | Milvus/Qdrant | RAG 向量存储 |
