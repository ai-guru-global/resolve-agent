## 什么是 ResolveAgent？

ResolveAgent 是一个面向问题解决的 AIOps 智能体平台，通过四大核心能力协同工作：

- 🔧 **专家技能** - 可插拔的领域专业知识
- 🌳 **FTA 工作流** - 故障树分析诊断
- 📚 **RAG 知识库** - 检索增强生成
- 💻 **代码分析** - 静态代码分析

## 为什么选择 ResolveAgent？

### 国产大模型支持

原生支持国内主流大模型：
- 通义千问 (阿里云)
- 文心一言 (百度)
- 智谱清言 (智谱 AI)

### 云原生架构

- 基于 Kubernetes 部署
- 使用 Higress 作为 AI 网关
- 支持 Helm 和 Kustomize

### 开源友好

Apache 2.0 许可证，欢迎社区贡献！

## 快速开始

```bash
git clone https://github.com/ai-guru-global/resolve-agent.git
cd resolve-agent
make setup-dev
make compose-up
```
